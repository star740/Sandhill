# Sandhill HQ — Company Website

Static site for Sandhill HQ: outsourced accounting & finance for SaaS startups and growing businesses.

Hosted on GitHub Pages. Maintained with Claude.

## Structure
- `index.html` — Home
- `services.html` — Services (three pillars + how we work)
- `who-we-help.html` — Who We Help / FAQ
- `about.html` — About Us + founder letter
- `blog.html` — Insights index
- `blog/` — Individual articles
- `contact.html`, `legal.html`, `404.html`
- `style.css` — shared styles
- `generate.py` — regenerates all pages from one template (edit this, run it, commit)
